# moon-gravity-simulation
A simple physics simulation of motion under lunar gravity using Python
![Result](image.png)
## 🌍 Comparison with Earth

On Earth, gravity is 9.8 m/s², which is much stronger than the Moon (1.62 m/s²).

This means:
- Jumps are higher on the Moon
- Motion lasts longer
- Objects fall more slowly

This simulation helps visualize these differences.
